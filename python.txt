Python es un lenguaje de programación interpretado, interactivo y orientado a objetos. Incorpora módulos, excepciones, tipado dinámico, tipos de datos dinámicos de muy alto nivel y clases. Soporta múltiples paradigmas de programación más allá de la programación orientada a objetos, como la programación procedimental y funcional. Python combina una potencia notable con una sintaxis muy clara. Dispone de interfaces para muchas llamadas al sistema y bibliotecas, así como para varios sistemas de ventanas, y es extensible en C o C++. También puede utilizarse como lenguaje de extensión para aplicaciones que necesiten una interfaz programable. Por último, Python es portátil: funciona en muchas variantes de Unix, incluidos Linux y macOS, y en Windows.
